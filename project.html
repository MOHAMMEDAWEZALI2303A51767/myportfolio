🎯 Project Overview

This website was created as part of a college project to showcase Synergy Fest 2025, our annual cultural and technical festival. It demonstrates the use of modern web design with responsive layouts, clean UI, and interactive sections like schedules and galleries.

👨‍💻 Features Implemented

  • Responsive design using HTML5 & CSS3
  • Event schedule with styled tables
  • Photo gallery with hover effects
  • Reusable external stylesheet
  • Simple navigation between pages

🚀 Future Improvements

In future versions, we plan to add online registration forms, live updates, and social media integration.
